# Data license & attribution

The **code** in this repository is MIT-licensed (see `LICENSE`).

The **map data** (the generated TopoJSON in `topojson/`) is derived from
[`reuters-graphics/country-borders`](https://github.com/reuters-graphics/country-borders),
which is built from [Overture Maps Foundation](https://overturemaps.org/) data
(released under the [ODbL](https://opendatacommons.org/licenses/odbl/)) with
supplementary geometry from [OpenStreetMap](https://www.openstreetmap.org/copyright)
contributors.

The generated TopoJSON is therefore an **ODbL derivative database**. Any use or
redistribution must preserve attribution and comply with the ODbL's
share-alike terms.

> © OpenStreetMap contributors, Overture Maps Foundation
